# 🃏 Aydni Poker

An online poker game platform built with Node.js, Express, and Socket.IO for real-time multiplayer gaming.

## Features

- 🎮 Real-time multiplayer poker gameplay
- 🔄 Live socket connections for instant updates
- 💰 Betting system
- 👥 Multiple players support
- 📱 Responsive design
- 🎨 Modern UI with poker table theme

## Tech Stack

- **Backend**: Node.js, Express.js
- **Real-time**: Socket.IO
- **Frontend**: HTML, CSS, JavaScript
- **Server**: HTTP

## Installation

1. Clone the repository:
```bash
git clone https://github.com/gombouuganaa0305-cloud/Aydni.mn.git
cd Aydni.mn
```

2. Install dependencies:
```bash
npm install
```

3. Start the server:
```bash
npm start
```

For development with auto-reload:
```bash
npm run dev
```

4. Open your browser and navigate to:
```
http://localhost:3000
```

## Usage

1. Open the website in your browser
2. Click "Join Game" to join a poker table
3. Place bets using the input field
4. Use Check or Fold buttons to play
5. Play with other online players in real-time

## Game Controls

- **Join Game**: Enter the game table
- **Check**: Pass without betting
- **Fold**: Discard your hand
- **Bet**: Place your wager
